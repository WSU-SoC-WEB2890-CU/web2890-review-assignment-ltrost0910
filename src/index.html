About

After retiring from a 35+ year career as a developer for enterprise systems focusing on backend processing and core business applications, I decided to return to Weber State to learn more about web design and development. Since then, I have expanded on my original WSU Information Technology bachelor's degree by adding a Web Essentials Certificate and will soon complete my AAS in Web and User Experience. I enjoy taking one class per semester as a retirement hobby and using my newly acquired web skills to volunteer with local groups and small businesses.
